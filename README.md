# Proyecto de Estudio de la Salud Mental por Países

Este proyecto analiza datos relacionados con la salud mental a nivel global, con un enfoque particular en la prevención del suicidio. Se han empleado herramientas de análisis estadístico y visualización para identificar patrones y correlaciones clave.

## Estructura del Proyecto

El proyecto está desarrollado en R Markdown y genera salidas en formatos HTML, Word y PDF. A continuación, se detalla la estructura:

1. **Información del Dataset**
   - Introducción al dataset utilizado.
   - Justificación de su selección.
   - Objetivos y pasos para alcanzarlos.

2. **Preparación del Entorno**
   - Configuración de paquetes requeridos: `ggplot2` y `corrplot`.

3. **Análisis Exploratorio de Datos**
   - Limpieza y preparación de los datos.
   - Visualizaciones iniciales para identificar tendencias.

4. **Análisis Avanzado**
   - Uso de herramientas de visualización como mapas de correlación.
   - Interpretación de patrones clave.

5. **Resultados y Conclusiones**
   - Discusión de hallazgos relevantes.
   - Propuestas para futuras investigaciones.

## Tecnologías Utilizadas

- **Lenguaje**: R.
- **Paquetes**: `ggplot2`, `corrplot`, entre otros.
- **Formato**: R Markdown, generando salidas interactivas y estáticas.

## Requisitos

- Tener instalado R y RStudio.
- Paquetes necesarios: `ggplot2`, `corrplot` (se instalan automáticamente si no están disponibles).

## Ejecución del Proyecto

1. Clonar el repositorio.
2. Abrir el archivo `.Rmd` en RStudio.
3. Ejecutar los chunks de código para generar las visualizaciones y resultados.
